# kcMeditate

A simple Meditation timer, designed and deployed as a progressive web app.

The timer app can be viewed online at:

**https://caseylabs.com/apps/simple-meditation-timer/**

![Demo](./demo.png)

## Local development

Serve the app:

```sh
make serve
```

Then open `http://localhost:8000`.

## Tests

```sh
make test
```
